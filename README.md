# Site Secretary — نظام سكرتير الموقع

## 🚀 التشغيل السريع

1. فُك ضغط الملف في أي مكان على ويندوز
2. اضغط دبل-كليك على **`RUN.bat`**
3. النظام سيفتح تلقائياً في نافذة Electron (أو المتصفح)

## 📋 المتطلبات

- ويندوز 10/11 (64-bit)
- Node.js 18+ (مُوصى به) — للتحميل: https://nodejs.org/
- أو Electron (يُفضّل) — يمكن وضعه بجانب `RUN.bat`

## 🎯 الميزات الرئيسية

### ✅ إدارة الترانسميتالات
- إنشاء / تعديل / حذف / نسخ الترانسميتالات
- ترقيم تلقائي مستقل لكل قسم
- تتبع المراجعات (REV.0 - REV.7)
- سير عمل الاستشاري والوزارة (MOH)

### ✅ نظام الترجمة الكامل (عربي / إنجليزي)
- زر تبديل اللغة 🇬🇧 / 🇸🇦 في الأعلى
- كل النصوص تُترجم فوراً
- الاتجاه يتغير تلقائياً (RTL / LTR)
- كل قسم / تخصص / نوع مستند له اسم عربي وإنجليزي

### ✅ الأقسام الرئيسية (5 أقسام جاهزة)
- 📄 ترانسميتال (Transmittal)
- 📦 تفتيش مواد (MIR)
- ✅ تشيك ليست (Check List)
- ❓ طلب معلومات (RFI)
- 📚 كتب (Letters)

### ✅ التخصصات الفرعية (22 تخصص جاهز)
- **ترانسميتال:** مدنية، كهربائية، صحي، تكييف، حريق، مصاعد
- **MIR:** مواد مدنية، كهربائية، صحية، تكييف، حريق
- **تشيك ليست:** مدنية، كهربائية، ميكانيكية
- **RFI:** مدنية، كهربائية، صحي، تكييف، حريق، مصاعد
- **كتب:** واردة، صادرة

### ✅ مسار الحفظ
- كل ترانسميتال له مجلد خاص به: `storage/uploads/{transmittalId}/`
- الملفات منظمة ويسهل الوصول إليها

### ✅ التعديل والحذف
- تعديل كامل لبيانات الترانسميتال (المرجع، التخصص، النوع، العنوان، الوصف)
- حذف مع تأكيد بكتابة المرجع بالضبط

### ✅ التقارير
- جدول زمني شامل لكل المراجعات
- طباعة (A4 landscape)
- تصدير Excel

### ✅ الرفع والتنزيل
- رفع صور (PNG, JPG, GIF, WebP) و PDF و Word
- تنزيل الملفات ببيانات نظيفة (base64)
- معاينة الصور inline

## 🔧 استكشاف الأخطاء

### المشكلة: النافذة لا تفتح
- تأكد من تثبيت Node.js
- شغّل `RUN.bat` كمسؤول (Right-click → Run as administrator)

### المشكلة: "Port 3000 in use"
- أغلق أي تطبيق آخر يستخدم المنفذ 3000
- أو عدّل `SERVER_PORT` في `electron-main.js`

### المشكلة: قاعدة البيانات لا تُفتح
- تأكد من وجود مجلد `db/` بصلاحيات كتابة
- إذا استمرت المشكلة، احذف `db/custom.db` وسيُنشأ تلقائياً
