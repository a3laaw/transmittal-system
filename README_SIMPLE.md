# Site Secretary — نسخة سهلة

**ديسك توب (.exe)**

خطوات التنصيب (`ويندوز` فقط):

1. فك ضغط الملف `.zip`
2. اضغط `INSTALL.bat` (مزدوج كليك)
3. بعد ما يخلص (`npm install` + `npm run build` + `.exe`)
4. افتح مجلد `dist` -> هتلاقي `.exe` -> اضغط عليه -> يتنصّب

- الاسم الأصلي (`file.name`) محفوظ.
- جدول مسطح (`mergeCells` محذوف).
- مسار ديناميكي (`uploads/{cat}/{disc}/{id}/`).
